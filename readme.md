# Laravel Sample

Laravel のサンプル 

## 開発環境構築

#### アプリケーションの構築
```
# 初回
make build

# server を起動
make serve

# composer
make composer

# assets compiling
make yarn-dev
```

https://localhost:8000 にアクセス

## 技術スタック
* Docker
* docker-compose
* Laravel 5.6.*
* PHP 7.2.*
* Laravel Mix
  * node:9
  * yarn
