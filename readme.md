# Laravel Sample

Laravel のサンプル

## 開発環境構築

#### アプリケーションの構築

```
# 初回
make build

# server を起動
make serve

# composer
make composer

# assets compiling
make yarn-dev
```

http://localhost:8000 にアクセス

## 技術スタック

-   Docker
-   docker-compose
-   Laravel 6.x
-   PHP 7.3
-   Laravel Mix
    -   node:12
    -   yarn
